# GitHub = GoFish_PS_SD
================= README FOR Go_Fish ==================

After unzipping GOFISH, go into the directory in which you unzipped the files
And type "make".

The following files will show up:
Card.h
Card.cpp
Deck.h
Deck.cpp
Player.h
Player.cpp
go_fish.cpp
Makefile
README.txt

After typing make, run the executable with this terminal command "./test".
After typing that executable, type "ls" to make sure that the output file is
in the directory. The output file name is "gofish_results.txt".
Open that file and you will find the results and the winner!
